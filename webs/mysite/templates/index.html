Several federal government agencies have photo repository websites which are isolated silos. This is a project to consolidate them in the interest of making these public domain images easier to find and use.

The Usable Image Scraper Library

A python library for maintaining a machine-readable mirror of several image repository websites. Adding a new image repository website takes as little work as possible--simply articulate the data schema and and write a parser for the image pages, and Usable Image Scraper Library will take care of the rest. Easy-to-use tests tell you which pieces are missing.
